# Fake Window Component

## Functionality

A mock desktop window, designed in the style of Windows 95.

## Props

`program`

## Props Functionality

### `program`

`<FakeProgram>`

Object containing data for a mock Windows 95 program.

- name (string): name of the file, shown in the topbar
- data (string): data to be displayed in the window
- type (string): type of file, determines window styling and functionality. acceptable options - png, jpg, jpeg, html, com
- open (boolean): open status of the file
- active (boolean): active status of the file window
- minimized (boolean): minimized status of the file window
