# Fake Start Menu Component

## Functionality

A mock start menu, designed in the style of Windows 95 icons.

## Props

`program`

## Props Functionality

### `programs`

`<FakeProgram>`

Array of objects containing data for a mock Windows 95 program.

- name (string): name of the file, shown in the button
- open (boolean): open status of the file
- active (boolean): active status of the file window
- minimized (boolean): minimized status of the file window
