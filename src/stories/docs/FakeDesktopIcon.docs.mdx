# Fake Desktop Icon Component

## Functionality

A mock desktop icon, designed in the style of Windows 95 icons.

## Props

`program`

## Props Functionality

### `program`

`<FakeProgram>`

Object containing data for a mock Windows 95 program.

- title (string): title of the file, shown underneath the icon
- type (string): type of file, determines icon image. acceptable options - png, jpg, jpeg, html, com
- open (boolean): open status of the file
- active (boolean): active status of the file window
