# Fake Desktop Icon Component

## Functionality
A mock desktop icon, designed in the style of Windows 95 icons.

## Props
`program`

## Props Functionality
### `program`
`<FakeProgram>`

Object containing data for a mock Windows 95 program.
- title: title of the file, show underneath the icon
- type: type of file, determines icon image. acceptable options - png, jpg, jpeg, html, com
